# report-parser

tl;dr: a simple parser for a daily HR report, which outputs the summary of employees on sick leave for each department/cost center.

I use it in conjunction with a BluePrism bot, which gets the HR report from the system, pastes it in the parser then copies the output and posts the results on a dashboard.

You can see it in action here: https://horiasimu.github.io/report-parser/

The input must be in the specific format outputted by our HR system, but for testing you can use the placeholder sample below.
